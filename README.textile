h1. open_tortoise_svn (Firefox Addon to open TortoiseSVN directly)

h2. Overview

* This Firefox Addon enables you to open TortoiseSVN directly when you click the link to the registered URLs.
* If the link has some specific attributes, this addon opens TortoiseSVN Repository Browser, Log Viewer or Blame Viewer.

h2. How to use

h3. Prepare to use

# Open setting dialog and set the path of TortoiseProc.exe.
# Add URLs of Subversion repositories that you want to open in TortoiseSVN.

For example, if you add the URL "http://example.net/svn_repos/" and click the link to "http://example.net/svn_repos/some_dir/", TortoiseSVN Browser is opened automatically.
<pre>&lt;a href="http://example.net/svn_repos/some_dir/"&gt;Open TortoiseSVN&lt;/a&gt;</pre>

h3. How to open TortoiseSVN Log/Blame View

You have to modify your HTML source and add specific class name and rel attribute to open TortoiseSVN Log or Blame view as follows.
You can use "data-tsvn-info" on HTML5 instead of "rel" attribute.

# To open TortoiseSVN Log view
<pre>&lt;a href="http://example.net/svn_repos/some_dir/" class="open_tsvn_link" rel="tsvn[log]"&gt;Open Log view&lt;/a&gt;</pre>
# To open TortoiseSVN Log of specific revision
<pre>&lt;a href="http://example.net/svn_repos/some_dir/" class="open_tsvn_link" rel="tsvn[log][1004,1000]"&gt;Open Log view&lt;/a&gt;</pre>
# To open TortoiseSVN Blame view
<pre>&lt;a href="http://example.net/svn_repos/some_dir/" class="open_tsvn_link" rel="tsvn[blame]"&gt;Open Blame view&lt;/a&gt;</pre>
# To open TortoiseSVN Browser
<pre>&lt;a href="http://example.net/svn_repos/some_dir/" class="open_tsvn_link" rel="tsvn[browser]"&gt;Open TortoiseSVN Browser&lt;/a&gt;</pre>
