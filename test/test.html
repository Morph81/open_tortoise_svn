Open TortoiseSVN Test

Prepare for test

Open Open TortoiseSVN Setting dialog, then add the following URLs.

  • http://localhost/
  • file:///HOME/

Add extension action as follows:

  • *.html, *.htm: Repository Browser
  • *.cpp: Log
  • *.c: Blame
  • *.txt: Open in Firefox

Test

The following links are not opened in TortoiseSVN.

  • http://example.net/ is an unregistered URL.
  • http://localhost:3000/ is an unregistered URL because port number is wrong.
  • href="http://localhost/" rel="tsvn[unknown]" is not opened in TortoiseSVN because rel attribute has an unknown action, unknown.
  • href="http://example.net/" rel="tsvn[browser]" is not an unregistered URL.

The following links are opened in TortoiseSVN repository browser.

  • http://localhost/svn/ is a registered URL.
  • href="file:///HOME/" rel="tsvn[browser]" is a registered URL and has correct rel attribute.
  • href="file:///HOME/" rel="tsvn[browser][]" is a registered URL and has correct rel attribute.
  • href="file:///HOME/" rel="tsvn[browser][2]" is a registered URL and has correct rel attribute.

The following links are opened in TortoiseSVN log browser.

  • href="file:///HOME/" rel="tsvn[log]" is a registered URL and has correct rel attribute.
  • href="http://localhost/" rel="tsvn[log][]" is a registered URL and has correct rel attribute.
  • href="file:///HOME/" rel="tsvn[log][2]" is a registered URL and has correct rel attribute. This link shows logs from revision 2.
  • href="file:///HOME/" rel="tsvn[log][2,1]" is a registered URL and has correct rel attribute. This link shows logs from revision 1 to 2.

The following links are opened in TortoiseSVN blame browser.

  • href="file:///HOME/" rel="tsvn[blame]" is a registered URL and has correct rel attribute.
  • href="file:///HOME/" rel="tsvn[blame][]" is a registered URL and has correct rel attribute.

  • HTML file: test.html should be opened in Repository Browser.
  • test.htm should be opened in Repository Browser.
  • test.cpp should be opened in Log..
  • test.c should be opened in Blame.
  • test.txt should be opened in Firefox.
