Open Open TortoiseSVN Setting dialog, then add the following URLs.

Test

The following links are not opened in TortoiseSVN.

  • http://example.net/ is an unregistered URL.
  • http://localhost:3000/ is an unregistered URL because port number is wrong.
  • href="http://localhost/" rel="tsvn[unknown]" is not opened in TortoiseSVN because rel attribute has an unknown action, unknown.
  • href="http://example.net/" rel="tsvn[browser]" is not an unregistered URL.

The following links are opened in TortoiseSVN repository browser.

  • http://localhost/svn/ is a registered URL.
  • href="http://localhost/" rel="tsvn[browser]" is a registered URL and has correct rel attribute.
  • href="http://localhost/" rel="tsvn[browser][]" is a registered URL and has correct rel attribute.

The following links are opened in TortoiseSVN log browser.

  • href="http://localhost/" rel="tsvn[log]" is a registered URL and has correct rel attribute.
  • href="http://localhost/" rel="tsvn[log][]" is a registered URL and has correct rel attribute.
  • href="http://localhost/" rel="tsvn[log][10]" is a registered URL and has correct rel attribute. This link shows logs from revision 10.
  • href="http://localhost/" rel="tsvn[log][10,3]" is a registered URL and has correct rel attribute. This link shows logs from revision 3 to 10.

The following links are opened in TortoiseSVN blame browser.

  • href="http://localhost/" rel="tsvn[blame]" is a registered URL and has correct rel attribute.
  • href="http://localhost/" rel="tsvn[blame][]" is a registered URL and has correct rel attribute.
